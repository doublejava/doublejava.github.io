# 侧边栏介绍

官网介绍

# themeConfig

找到config.js中的themeConfig，关注sidebarDepth 这个值。 目前最大深度是2。也就是当设置为2时，会自动加载h2、h3作为标题，显示在侧边栏下。 设置为1则显示h2，设置为0，则屏蔽。

# sidebar 侧边栏设置

//侧边栏配置
    sidebar: {
      '/vuepress/': [
        '/vuepress/', //显示侧边栏顶部那块，对应h2 h3 ，sidebarDepth 这个值
        {
          title: 'vuepress搭建',
          children: [
            '/vuepress/step/one',
            '/vuepress/step/two',
            '/vuepress/step/three'
          ]
        }
      ]
    }
